## 👾 Space Shooter — Contributions as Targets

<div align="center">
  <img src="./game.gif" width="90%" alt="Space Shooter game generated from contribution graph"/>
</div>

> *This turns your contribution squares into targets in an animated space shooter. It needs a one-time GitHub Action setup (below) — until then, this section will just show a broken image, so add the workflow first.*

<details>
<summary><b>⚙️ Setup: enable the Space Shooter game</b></summary>

<br>

In your `Madathanapalleleena/Madathanapalleleena` repo, create `.github/workflows/update-game.yml`:

```yaml
name: Update Space Shooter Game
on:
  schedule:
    - cron: '0 0 * * *'   # daily at midnight UTC
  workflow_dispatch:       # allows manual run from the Actions tab
permissions:
  contents: write
jobs:
  update-game:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v6
        with:
          fetch-depth: 2
      - uses: czl9707/gh-space-shooter@v2
        with:
          github-token: ${{ secrets.GITHUB_TOKEN }}
          output-path: 'game.gif'
          strategy: 'random'
```

Commit this file, then go to the **Actions** tab and manually run it once (via "Run workflow") so `game.gif` gets generated immediately instead of waiting for midnight. Once it runs successfully, the `./game.gif` reference above will render.

</details>
